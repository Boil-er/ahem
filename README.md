# *Ahem*

This application is a simple talk planning tool tailored for JW talks.

## Goals

*Ahem* is meant to make preparing and giving a talk easier by:
* providing a writing environment designed around how JW talks are actually built
* producing a printable plan that is easy to scan mid-talk, unlike plain text plans
* encouraging thoughtful preparation through talk metadata
* being an open and free alternative to generic word processors

## Limitations

*Ahem*:
* is a hobby project by a solo developer and thus will not have enterprise grade maintenance or development speed
* may be inherently biased toward a specific workflow or plan style (again, solo developer)
* does not save the data in cloud and is thus more fragile by nature than some more mature and cloud-based alternatives
* requires more management effort from the user because they need to save and version their talks themselves
* requires a decent color printer to benefit from the printable export feature
* does not boost talk quality on its own - it is a tool, not a guide/assistant
* currently expects more technical know-how from the user than some alternatives — intending to improve this over time

## Author's note

The author is a Jehovah's Witness but *Ahem*:
* is NOT affiliated with or endorsed by Watch Tower Society in any way (i.e. is NOT an official JW app)
* does NOT provide any spiritual food or teach the user how to prepare a good talk
* is fully free and open-source
* keeps all data on user's device, at all times - nothing is ever sent to any server or database
* advises the user to export their talk to their own storage after each session — browser storage can be cleared unexpectedly
